# Examples
## log 보기
```sh
$ git log --since=2.weeks   # 오늘부터 2주 전까지의 커밋 로그를 본다
$ git log -S test           # 변경사항 중 test 라는 문자열이 있는 커밋을 검색
$ git log --grep test       # test 를 포함하는 커밋 메시지를 가진 커밋을 검색
```
